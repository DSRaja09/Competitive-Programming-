# Competitive Programming Solutions

Welcome to my personal collection of algorithms and data structures. This repository is dedicated to efficient, battle-tested code templates for competitive programming contests.

## 📁 Topics Covered

| Category | Key Concepts Included |
| :--- | :--- |
| **Number Theory** | Modular Inverse, Sieve, GCD/LCM, Euler's Totient, Lucas Theorem |
| **Graph Theory** | BFS/DFS, Dijkstra, Kruskal's/Prim's, LCA, SCC, Bipartite Matching |
| **Dynamic Programming** | Knapsack, LIS, LCS, Digit DP, Bitmask DP, DP on Trees |
| **Segment Tree** | Point Update, Range Query, and **Lazy Propagation** |
| **Combinatorics** | Permutations/Combinations, Inclusion-Exclusion, Catalan Numbers |
| **String Hashing** | Rolling Hash, Collision avoidance, Substring comparison |

---

## 🛠 Features

### Segment Tree with Lazy Propagation
Implementation supports $O(\log n)$ range updates and range queries. It is designed to handle common operations like range sum, min/max, and range assignment.

### String Hashing
Utilizes double-hash techniques to ensure minimal collision probability during string matching and palindrome detection.
